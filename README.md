# adamzou.fr — Hub Portfolio

> Fait avec Claude par Adamzou — Holberton School AI Engineer · 2025

---

## L'idée

Créer un hub central qui regroupe tous mes projets, mon CV et mes coordonnées — une vitrine personnelle moderne, accessible depuis un nom de domaine unique : **adamzou.fr**.

L'idée : un design glassmorphism élégant, sobre, avec une palette verte émeraude, entièrement responsive et optimisé pour le SEO.

---

## Le cheminement

Ce projet est né d'une collaboration entre une vision humaine et une exécution assistée par IA.

**1. Le concept** — L'idée, la palette de couleurs (vert émeraude `#0a7a55`), la structure du hub et les exigences de design viennent d'Adamzou. Un hub unique qui centralise projets, stack technique, et contact.

**2. La génération initiale** — Le code HTML/CSS/JS a été généré via Claude (claude.ai) à partir d'un prompt précis définissant le design glassmorphism, la police Outfit, la palette de couleurs et les sections attendues.

**3. L'optimisation SEO** — En collaboration avec Claude Code (CLI), une analyse SEO complète a été menée : ajout des balises `<meta description>`, `canonical`, Open Graph complet, Twitter Card, JSON-LD Person schema (schema.org), `robots.txt` et `sitemap.xml`.

**4. L'amélioration structurelle** — Analyse et refonte de la structure : ajout de `<header>`, `<main>`, `<nav>`, `<footer>` sémantiques, remplacement de l'obfuscation email Cloudflare par un vrai lien `mailto:`, styles inline déplacés en classes CSS, JS encapsulé dans une IIFE.

**5. L'intégration des projets** — Chaque projet (nowindcity, MyCal) a été ajouté sous forme de carte avec lien live, lien GitHub, tags de stack technique et badge de statut.

**6. La stack technique** — Mise à jour pour refléter les compétences réelles : HTML5, CSS3, Git & GitHub, Linux, C (en cours d'apprentissage à Holberton School).

---

La direction créative, le nom de domaine, la structure du hub, les choix de design et les décisions de contenu viennent d'Adamzou. Claude a été l'outil d'exécution et d'analyse.

## Stack technique

- HTML5 / CSS3 / JavaScript ES6+ (vanilla, aucun framework)
- Design : glassmorphism, palette verte émeraude `#0a7a55`
- Police : Outfit (Google Fonts)
- SEO : Open Graph, Twitter Card, JSON-LD schema.org
- Aucune dépendance externe

---

## Fonctionnalités

- Hero section avec présentation et CTA
- Section projets avec cartes glassmorphism
- Section stack technique
- Section contact (email, GitHub, LinkedIn)
- 100% responsive — mobile first
- Animations au scroll (Intersection Observer)
